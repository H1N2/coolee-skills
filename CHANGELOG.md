# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-02-07

### Added
- **coolee-leijun-speech**: New content creation skill that transforms topics or text into Lei Jun-style keynote speeches
  - Data-driven narratives with precise numbers and comparisons
  - Grand storytelling with emotional resonance
  - Professional terminology creation
  - Three-act structure (problem → solution → value reveal)
  - Signature Lei Jun catchphrases and sentence patterns
  - Master prompt template for consistent output
  - Complete example with smart desk lamp presentation
  - Supports both Chinese and English inputs

### Changed
- Enhanced README.md with detailed skill descriptions, installation guide, and project structure
- Updated project documentation to follow open-source best practices

### Fixed
- Improved SKILL.md documentation with bilingual (Chinese/English) terminology
- Added clear usage instructions for all skills

## [0.1.0] - 2026-02-07

### Added
- Initial project structure based on baoyu-skills conventions
- **coolee-claudian-updater**: Utility skill for automatically updating Obsidian's Claudian plugin
  - Auto-detects Obsidian vault and plugin location
  - Version checking against GitHub releases
  - Automatic backup before updates
  - Safe installation with rollback capability
- Project guidelines in CLAUDE.md
- MIT License
- Basic README and documentation structure
