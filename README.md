# Coolee Skills

A collection of efficiency skills for Claude Code, mirroring the structure and standards of `JimLiu/baoyu-skills`.

## Installation

To use these skills with Claude Code:

1. Clone this repository.
2. Follow the Claude Code documentation to add local skills.

## Skills

### Utility
- **coolee-claudian-updater**: Automatically update the Claudian plugin for Obsidian.
