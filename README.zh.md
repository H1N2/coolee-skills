# Coolee Skills (中文)

本仓库是 Claude Code 的效率工具集，参考了 `JimLiu/baoyu-skills` 的结构和标准。

## 安装

在 Claude Code 中使用这些 Skill：

1. 克隆本仓库。
2. 按照 Claude Code 文档添加本地 Skill。

## 已包含的 Skill

### 工具类
- **coolee-claudian-updater**: 自动更新 Obsidian 的 Claudian 插件。
